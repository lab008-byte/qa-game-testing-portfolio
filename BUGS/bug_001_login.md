# Bug Report - Login Button Not Working

**ID:** BUG-001  
**Title:** Login button does not respond when clicked  
**Severity:** High  
**Priority:** Critical  
**Environment:** PC, Windows 10, Chrome  

## Description
The login button does not trigger any action after entering valid credentials.

## Steps to Reproduce
1. Open the application
2. Go to login screen
3. Enter valid username and password
4. Click "Login"

## Expected Result
User should be logged into the system.

## Actual Result
Nothing happens when clicking the button.
