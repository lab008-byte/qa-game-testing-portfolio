# Bug Report - UI Overlapping Elements

**ID:** BUG-002  
**Title:** UI elements overlap on small screens  
**Severity:** Medium  
**Priority:** High  
**Environment:** Android devices  

## Description
UI elements such as buttons and text overlap when displayed on smaller screens.

## Extended Description
This issue affects usability and readability, especially on low-resolution devices. It indicates a lack of responsive design handling and may result in users being unable to interact properly with the interface.

## Steps to Reproduce
1. Open the app on a small screen device
2. Navigate to main menu

## Expected Result
UI should scale properly and maintain spacing between elements.

## Actual Result
Text and buttons overlap, making content hard to read and interact with.
