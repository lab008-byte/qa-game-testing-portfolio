# QA Game Tester Portfolio - Bastian Lara

## Description
This repository contains my QA Game Tester portfolio. It includes found bugs, test cases, and testing scripts, including some developed in **Godot** for automation or simulation purposes. This project is for practice and to showcase skills in video game QA.

## Repository Structure
- **Bugs/** → Bug reports with detailed steps to reproduce and their impact on gameplay.  
- **Scripts/** → Test and automation scripts for various games or test scenarios.  
- **ScriptsGodot/** → Scripts developed in **Godot** Engine for in-game testing or simulations.  
- **Evidence/** → Screenshots, videos, or logs of executed tests (optional).  

## How to Use This Portfolio
1. Check the **Bugs/** folder to review the bugs found and their documentation.  
2. Explore **Scripts/** to see examples of automation or support scripts.  
3. Review **ScriptsGodot/** for Godot-specific testing scripts.  
4. If available, see **Evidence/** for visual results of tests.
