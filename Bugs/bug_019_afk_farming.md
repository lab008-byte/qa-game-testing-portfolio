# Bug Report - AFK Farming Exploit

**ID:** BUG-019  
**Game:** Generic Roblox Simulator  
**Title:** Players gain rewards while inactive  
**Severity:** Medium  
**Priority:** Medium  
**Environment:** Roblox / Progression  

## Description
Players continue earning rewards without active input.

## Extended Description
This allows players to progress without playing, affecting fairness and engagement.

## Steps to Reproduce
1. Start a reward-generating action
2. Stay idle
3. Observe rewards over time

## Expected Result
Rewards should require active participation.

## Actual Result
Rewards continue while player is inactive.

Open
