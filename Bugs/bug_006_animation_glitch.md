# Bug Report - Character Animation Glitch

**ID:** BUG-006  
**Title:** Character animation breaks during movement  
**Severity:** Medium  
**Priority:** Medium  
**Environment:** PC / Gameplay  

## Description
Character animations glitch or freeze when transitioning between movement states.

## Extended Description
During transitions (idle to run, run to jump), the animation may freeze, loop incorrectly, or snap unnaturally. This reduces visual quality and may impact gameplay feedback.

## Steps to Reproduce
1. Move character forward
2. Quickly jump or stop
3. Observe animation transition

## Expected Result
Smooth and natural animation transitions.

## Actual Result
Animation glitches or freezes momentarily.

## Status
Open
