# Bug Report - Player Clipping Through Walls

**ID:** BUG-007  
**Title:** Player can pass through walls  
**Severity:** Critical  
**Priority:** High  
**Environment:** Gameplay / Map collision  

## Description
The player can clip through certain walls or objects.

## Extended Description
This issue allows players to exit the intended map boundaries or access unintended areas, potentially breaking the game or enabling exploits.

## Steps to Reproduce
1. Move toward a wall at an angle
2. Jump or sprint into it
3. Observe collision behavior

## Expected Result
Player should be blocked by walls.

## Actual Result
Player passes through or partially clips into the wall.

## Status
Open
