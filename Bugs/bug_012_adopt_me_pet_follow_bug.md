# Bug Report - Pet Follow Behavior Issue

**ID:** BUG-012  
**Game:** Adopt Me!  
**Title:** Pet does not follow player correctly  
**Severity:** Low  
**Priority:** Medium  
**Environment:** Roblox / Mobile  

## Description
Pets stop following the player or lag behind significantly.

## Extended Description
This impacts the core gameplay experience since pets are a central feature. The issue may be related to pathfinding or network synchronization.

## Steps to Reproduce
1. Equip a pet
2. Start moving around the map
3. Observe pet behavior

## Expected Result
Pet should follow the player smoothly.

## Actual Result
Pet stops, lags, or teleports inconsistently.

## Status
Open
