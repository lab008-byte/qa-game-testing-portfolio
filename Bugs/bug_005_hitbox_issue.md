# Bug Report - Incorrect Hitbox Detection

**ID:** BUG-005  
**Title:** Enemy hitbox does not match visual model  
**Severity:** High  
**Priority:** High  
**Environment:** PC / Gameplay  

## Description
The enemy hitbox does not align with the visible model, causing attacks to register incorrectly.

## Extended Description
Players may hit the enemy visually without registering damage, or receive damage without proper contact. This creates unfair gameplay and affects combat accuracy.

## Steps to Reproduce
1. Engage with an enemy
2. Attack near the edge of the enemy model
3. Observe hit registration

## Expected Result
Hits should register accurately based on the visual model.

## Actual Result
Hit detection is inconsistent and does not match the model.

## Status
Open
