# Bug Report - Item Duplication Exploit

**ID:** BUG-015  
**Game:** Generic Roblox Economy Game  
**Title:** Items can be duplicated using trade interruption  
**Severity:** Critical  
**Priority:** Urgent  
**Environment:** Roblox / Multiplayer  

## Description
Players are able to duplicate items by interrupting a trade at a specific moment.

## Extended Description
This exploit allows players to generate unlimited items, breaking the in-game economy. It can lead to inflation, unfair advantages, and loss of game integrity.

## Steps to Reproduce
1. Start a trade with another player
2. Add an item to trade
3. Confirm trade
4. Disconnect or cancel at specific timing

## Expected Result
Item should only exist once and transfer correctly.

## Actual Result
Item remains in inventory and is also received by the other player.
