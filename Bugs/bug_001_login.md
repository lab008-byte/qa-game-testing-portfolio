# Bug Report - Login Button Not Working

**ID:** BUG-001  
**Title:** Login button does not respond when clicked  
**Severity:** High  
**Priority:** Critical  
**Environment:** PC, Windows 10, Chrome  

## Description
The login button does not trigger any action after entering valid credentials, preventing users from accessing the system.

## Extended Description
This issue blocks user access completely, making the application unusable for authenticated features. No feedback, loading indicator, or error message is shown, which may confuse users and lead to repeated attempts or abandonment.

## Steps to Reproduce
1. Open the application
2. Navigate to login screen
3. Enter valid credentials
4. Click "Login"

## Expected Result
User should be authenticated and redirected to the main dashboard.

## Actual Result
No action is triggered after clicking the login button.
