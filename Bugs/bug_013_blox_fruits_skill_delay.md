# Bug Report - Skill Activation Delay

**ID:** BUG-013  
**Game:** Blox Fruits  
**Title:** Skills activate with delay during combat  
**Severity:** High  
**Priority:** High  
**Environment:** Roblox / Combat  

## Description
Abilities have a delay between activation and execution.

## Extended Description
This negatively impacts combat responsiveness and player performance. Could be caused by server latency or scripting delays.

## Steps to Reproduce
1. Enter combat
2. Use any ability
3. Observe activation timing

## Expected Result
Skills should activate instantly.

## Actual Result
Noticeable delay before execution.

## Status
Open
