**ID:** BUG-004  
**Title:** Game crashes immediately after launch  
**Severity:** Critical  
**Priority:** Urgent  
**Environment:** PC, Windows 11  

## Description
The game crashes instantly after being launched.

## Extended Description
This is a critical blocking issue that prevents users from accessing the application entirely. Possible causes include missing dependencies, corrupted files, or incompatibility with the operating system.

## Expected Result

## Actual Result
